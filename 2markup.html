lists

There are three kinds of lists: ordered, unordered, and definition. List items are wrapped in a list container.

Ordered List

  1. Item one
  2. Second item
  3. Another item
  4. Last one

Unordered List

  • First same
  • Not again
    • Lists can be
    • Nested to
    • Arbitrary depth
  • Henceforth

Images

Images use the img tag, which is self-closing, and they might be inside a figure tag. Images use the src= attribute to point to the source, which could be anywhere. Images display inline by default.
